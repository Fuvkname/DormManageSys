# DormManageSys
学生信息管理系统

说明：
本系统是Java SSM架构搭建的一个比较小型的demo,前端使用到Bootstrap作为UI设计。
使用前请在Mysql导入sql文件进行数据库的构建
并在src的resource目录下修改jdbc.properties文件修改好数据库信息
使用idea导入为maven项目并导入相关的依赖以及tomcat插件运行（均已在pom.xml文件内配置好了）。
地址栏输入localhost:8848访问首页(端口号可以在pom.xml的Tomcat配置内设置)
具体账号密码可查询数据库内的user表以及admin表。

====================================================


本系统分为管理员和学生两个客户端
学生端的功能比较简单包括仅包括查询宿舍信息（包括水电费以及宿舍成员）。
管理员的功能包括对宿舍信息的增删查改以及缴费。

================================================================
